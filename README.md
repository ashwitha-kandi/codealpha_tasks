## 1. [Hangman Game](hangman/)

A simple Python-based Hangman game.

**Folder:** `hangman/`

### Features
- Classic Hangman gameplay in the browser.
- Dynamic word selection.
- Tracks guesses and displays the hangman visually.

### How to Run

cd hangman
pip install -r requirements.txt
python app.py
